# HDPE Pipes Landing Page

A fully responsive industrial landing page developed using **HTML5**, **CSS3**, and **Vanilla JavaScript**.  
The project follows the provided Figma design closely and includes smooth interactive UI components, responsive layouts, and modern frontend functionality.

---

## 🚀 Live Features

### ✅ Responsive Design
- Optimized for:
  - Desktop
  - Laptop
  - Tablet
  - Mobile Devices
- Fluid layouts with clean responsiveness
- No horizontal overflow issues

- Sticky navigation on scroll

- Smooth open/close animation
- Fully responsive behavior
